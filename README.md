#Finance Tracker

Finance Tracker is a desktop app built in Python for the backend and SQLite for the database. Tracks your expenses, your income, and reports monthly statistics. This application has a couple of pages, including
a dashboard page for general purposes, an import page for users to add their finances, a transaction page for users to see their transaction, and add any if needed, a budgets page for users to add their budgets,
a reports page to show their monthly or yearly reports, as well as a accounts and categories page. The application can be run by downloading the files and running either app.py or run.py.
